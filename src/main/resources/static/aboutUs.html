Dies ist eine auf html, css und js basierende Webaufzeichnungs- und Videoanwendung. Die Hauptfunktion ist die Aufzeichnung einer Webseite nach Wahl des Benutzers oder die Verwendung der Computerkamera, um ein Video aufzunehmen und es in den lokalen Bereich herunterzuladen. Sie kann auch lokale Videos hochladen und die Leinwand manipulieren, um das Video zu bearbeiten und lokal herunterzuladen. Mit der Screenshot-Funktion können Sie bis zu neun Bilder aus einem Video auf einmal aufnehmen und sie in großen Mengen herunterladen.
Diese Anwendung bietet die Möglichkeit, das Video auf den Server hochzuladen, indem zunächst eine Vorschau des Videos auf der Webseite angezeigt wird und dann das lokale Video auf den Server hochgeladen wird, basierend auf Java-Code.
